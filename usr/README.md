# Haramaya University Research App

An application built to present the ending year research project for the Accounting and Finance Department at Haramaya University.

## Project Details
- **Student:** Abdulfeta Usman
- **Advisor:** Adane Tlihun
- **Department:** Accounting and Finance
- **Institution:** Haramaya University

## Features
- **Research Presentation:** Clear structure for displaying chapters, methodologies, and findings.
- **Cross-Platform:** Works on mobile, desktop, and web environments out of the box.
- **Modern UI:** Clean Material Design layout.

## Tech Stack
- Flutter
- Dart

## Setup
To run this application locally:
1. Ensure Flutter is installed and configured.
2. Clone the repository and navigate to the project root.
3. Run `flutter pub get` to fetch dependencies.
4. Run `flutter run` to start the application.
